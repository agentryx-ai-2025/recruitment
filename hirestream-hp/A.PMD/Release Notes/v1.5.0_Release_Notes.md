# HireStream — v1.5.0 Release Notes

**Release date:** 2026-04-15
**Theme:** *Runtime-configurable super-portal + overseas-placement regulatory depth*

This release adds 21 runtime-configurable settings, an agent productivity suite, reports/BI, and full MEA/Emigration-Act regulatory tracking. Nothing in this release requires a code deploy to re-tune — everything is admin-switchable through the new **System Config** panel.

---

## 1. System Configuration Module (new)

Admin → **System Config** tab. 21 settings across 8 categories, each live-editable with no restart.

**UAT vs Production defaults:** ships in UAT-open mode. A 4-toggle pre-prod checklist is in the reference doc.

## 2. Agent productivity (new)

### Applicant pipeline (`/agent/jobs/:id`)
- **Internal notes** per applicant (collaborative team notes, visible only inside the agency).
- **Bulk actions** — select multiple applicants → Mark Reviewed / Shortlist All / Reject All.
- **Schedule interview modal** with date/time/mode/location + optional drive selector + auto-notify candidate.
- **CSV export** of all applicants (satisfies FRS 2.10 "Download applicant profiles" properly).

### Drive detail (`/agent/drives/:id`)
- **Structured interview feedback** — rating (1–5), strengths, concerns, recommendation (strong yes/yes/maybe/no/strong no), notes. Persisted per interview.
- **ICS calendar export** button — adds the interview to Google Calendar / Outlook / Apple Calendar.
- **Reversible interview results** — Selected / Hold / Rejected now re-editable, with Clear option.

### Candidate profile (`/agent/candidates/:id`)
- **Pre-Departure Compliance panel** (see section 3).
- **Post-placement welfare check-ins** (30 / 60 / 90 day).

### Agent dashboard overview
- **Reports tab** — jobs / applicants / placements / conversion-rate + applicant funnel + avg time-to-placement + top destination countries + applications-over-time trend.
- **Job Edit** — inline pencil on job card opens JobPoster in edit mode.
- **Recent Jobs + Recent Drives** cards now clickable.

## 3. Overseas-placement regulatory depth (new — beyond FRS)

Candidates now carry the full MEA / Emigration Act, 1983 compliance record that an overseas placement agency is expected to maintain:

| Field | Purpose |
|---|---|
| Passport # + expiry | Valid travel doc |
| ECR / ECNR status | Determines POE clearance need |
| PCC status + expiry | Police Clearance Certificate |
| Medical fitness | Fit / pending / unfit + date |
| IELTS band | For UK/Canada/AU/NZ placements |
| Language proficiency (JSON) | Flexible for Arabic/German/Japanese etc. |
| PDO completed + date | Pre-Departure Orientation (mandatory for ECR) |
| PBBY insurance status + policy # | Pravasi Bharatiya Bima Yojana (mandatory under Emigration Act §10) |

**Post-placement welfare** (MEA requirement): 30-day / 60-day / 90-day check-ins per placement. Status: ok / concerns / no_response / not_applicable + free-text notes.
