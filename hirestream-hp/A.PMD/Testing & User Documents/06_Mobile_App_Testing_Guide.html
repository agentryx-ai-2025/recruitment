📱 HireStream Mobile App

Step-by-step setup guide for HPSEDC testing team

Android · Expo Go · 5 minutes to set up
1
Install "Expo Go" from Google Play Store
Free app · ~60 MB · by Expo Project

Open the Google Play Store on your Android phone and search for "Expo Go".

👆 Action: Tap the green Install button and wait for the download to complete.

Expo Go is a free testing app used by developers worldwide. It allows you to preview mobile apps without installing separate APK files.

Search for "Expo Go" in the Play Store and tap Install
2
Open Expo Go and tap "Enter URL manually"
Skip the QR scanner for now

After installation, open the Expo Go app. You will see the welcome screen.

👆 Action: Tap the "Enter URL manually" link (below the QR code button).
⚠️ Note: You do NOT need to create an Expo account. Skip any sign-up prompts.
Tap "Enter URL manually" — highlighted with the yellow arrow
3
Enter the HireStream app URL
Type the exact URL shown below

In the URL input field, type the following address exactly as shown:

exp://hirestream-mobile.agentryx.dev:80
👆 Action: After typing the URL, tap the "Connect" button.
⚠️ Important: Make sure to type exp:// (not https://). Include the :80 at the end.
Type the URL and tap "Connect" — the arrow shows where to tap

🔲 Or scan this QR code instead

If you prefer, go back and tap "Scan QR code" in Expo Go, then point your camera at this code:

exp://hirestream-mobile.agentryx.dev:80
4
Wait for the app to load
First load takes 15–30 seconds

After tapping Connect, the app will download and compile. You will see a loading screen.

⏳ Wait: The first load takes 15–30 seconds. This is normal — subsequent loads are faster.
⚠️ If you see "Something went wrong": Pull down on the screen to refresh, or close Expo Go completely and reopen it, then re-enter the URL.
The loading screen — wait for it to finish
5
Login with your test credentials
Use any of the test accounts below

Once loaded, you will see the HireStream login screen. Enter your test credentials.

👆 Action: Enter username mobiletest and password test123, then tap Sign In.
✅ Success! You should now see the home screen with available job listings. You can browse jobs, apply, check notifications, and manage your profile.

🔧 Troubleshooting

App won't load / blank screen
Close Expo Go completely (swipe away from recent apps), reopen it, and re-enter the URL.
"Something went wrong" error
Pull down on the screen to refresh. If it persists, check your internet connection.
Login fails
Usernames are case-sensitive. Make sure to type exactly as shown (lowercase). Password is test123.
QR code doesn't work
Make sure Expo Go is installed first. Try the manual URL entry method instead (Step 2 → Step 3).
App loads very slowly
The first load downloads the JavaScript bundle (15–30 seconds). Subsequent loads are much faster.
