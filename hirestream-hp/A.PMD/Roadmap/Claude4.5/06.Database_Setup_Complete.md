## 🔍 Database Management

### Connect to Database
```bash
# As postgres superuser
sudo -u postgres psql -d hirestream

# As hirestream user
psql -U hirestream -d hirestream -h localhost
# Password: hirestream
```

### Common Commands
```sql
-- List all tables
\dt

-- View table structure
\d users
\d candidates
\d jobs
\d applications

-- Count records
SELECT COUNT(*) FROM users;
SELECT COUNT(*) FROM jobs;
SELECT COUNT(*) FROM candidates;

-- View all users
SELECT * FROM users;

-- Exit
\q
```

## 🛠️ Database Maintenance

### Backup Database
```bash
# Create backup
sudo -u postgres pg_dump hirestream > hirestream_backup_$(date +%Y%m%d).sql

# Restore backup
sudo -u postgres psql hirestream < hirestream_backup_20251130.sql
```

### View Logs
```bash
# PostgreSQL logs
sudo tail -f /var/log/postgresql/postgresql-16-main.log

# Application logs (database queries)
pm2 logs hirestream
```
