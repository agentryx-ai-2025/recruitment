## 🔧 Solution: Open Port 5000 in Cloud Firewall

### For Google Cloud Platform (GCP)

Since your IP is `34.131.128.67`, you're likely on GCP. Here's how to open the port:

#### Option 1: Using gcloud CLI (Fastest)

```bash
# Allow incoming traffic on port 5000
gcloud compute firewall-rules create allow-hirestream \
    --direction=INGRESS \
    --priority=1000 \
    --network=default \
    --action=ALLOW \
    --rules=tcp:5000 \
    --source-ranges=0.0.0.0/0 \
    --description="Allow HireStream application on port 5000"
```

#### Option 2: Using GCP Console (Web Interface)

1. Go to [GCP Console](https://console.cloud.google.com/)
2. Navigate to **VPC Network** → **Firewall**
3. Click **CREATE FIREWALL RULE**
4. Configure:
   - **Name**: `allow-hirestream` or `allow-port-5000`
   - **Direction**: Ingress
   - **Action on match**: Allow
   - **Targets**: All instances in the network
   - **Source filter**: IP ranges
   - **Source IP ranges**: `0.0.0.0/0` (allow from anywhere)
   - **Protocols and ports**: Check **tcp** and enter `5000`
5. Click **CREATE**
