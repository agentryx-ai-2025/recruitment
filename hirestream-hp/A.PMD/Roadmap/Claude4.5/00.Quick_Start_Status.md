## 🔥 Quick Fix - Open Port 5000

### Method 1: GCP Console (Recommended - 2 minutes)

1. **Open GCP Console**: https://console.cloud.google.com/
2. **Go to Firewall**: Navigation menu → VPC Network → Firewall
3. **Click**: "CREATE FIREWALL RULE"
4. **Fill in**:
   ```
   Name: allow-hirestream-5000
   Direction: Ingress
   Action: Allow
   Targets: All instances in network
   Source IP ranges: 0.0.0.0/0
   Protocols/Ports: tcp:5000
   ```
5. **Click**: "CREATE"
6. **Wait**: 30 seconds for rule to apply

### Method 2: gcloud CLI (If authenticated)

```bash
gcloud auth login  # First authenticate
gcloud config set project YOUR_PROJECT_ID

# Then create rule
gcloud compute firewall-rules create allow-hirestream-5000 \
    --direction=INGRESS \
    --action=ALLOW \
    --rules=tcp:5000 \
    --source-ranges=0.0.0.0/0
```
