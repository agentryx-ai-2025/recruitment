Part 2 · User & Testing Guide

How the portal works

The workflow — especially for candidates — has changed a lot. This guide walks through the new candidate experience, the HPSEDC super-agency, and the admin console, with test steps for each.

Access & roles

Candidate (Job Seeker)Register on the portal (password needs one capital + one special character, e.g. Test@1234). Anyone can register.
HPSEDC AgencyDemo account hpsedc_agency (quick-login). The single super-agency — owns all jobs, candidates & placements.
AdminDemo account demo_admin (quick-login). Oversight, callbacks, support queue, grievances.
The portal is bilingual — use the language toggle in the top bar to switch English ⇄ Hindi.

Candidate · three ways to register

Applicants range from low-literacy blue-collar workers to degree-holding professionals, so registration offers three modes:

A · Standard — "Fill it myself (quick form)"

For blue-collar workers. A picture-based, one-question-per-screen form (~2 min) with voice input — completable by the worker or a helper (family / Common Service Centre).

B · Professional — "I have a degree or professional qualification"

For higher-education applicants (nurses, engineers, IT). A guided form capturing education → degree → field → institution, certifications, IELTS & passport.

C · Assisted — "Ask HPSEDC to call me"

The minimum-information path: leave only name + phone; HPSEDC calls back and completes it. Configurable — HPSEDC can switch this option on/off from the backend.

The three registration modes shown to a new candidate.

Standard (blue-collar) & Professional flows

Standard — pictorial, one question per screen.
Professional — guided, detailed capture.

The dashboard · Minimal ⇄ Advanced

Everyone uses the same dashboard layout. Blue-collar applicants get a Minimal view (few big buttons); professionals get the Advanced view (full menu + stats). A toggle switches between them.

Minimal — blue-collar default.
Advanced — professional default.

Documents are required

A profile is not "ready" until an ID (Aadhaar / Voter ID) and a passport are uploaded — mandatory for overseas placement. Photo or PDF accepted. Once done, the dashboard shows "Your profile is ready" with the HPSEDC registration number.

Documents — ID + passport required.
Profile ready + registration number.

Find a job → apply → track

Each job shows the typical monthly pay for that category & country and the documents that country requires (Have-it / Upload / Arrange).

Job detail — salary band + country-specific required documents.
My Application — Applied → Review → Interview → Selected.
Post-placement support + monthly check-in + MADAD / eMigrate.

HPSEDC · the single super-agency

Instead of many competing agencies, HPSEDC is the one super-agency: all jobs belong to HPSEDC, which manages every candidate and placement and runs the full pipeline Submitted → Reviewed → Shortlisted → Interview → Selected → Placed. At Selected a placement record (visa / travel / welfare) is auto-created; the application becomes Placed when the candidate accepts the offer.

Agency view — candidate, compliance (MEA), visa / travel & placement management.

Admin console

Admin keeps HPSEDC oversight — overview, Callbacks (Assisted-tier requests), Placement Support queue, grievances, reports and settings. Features carried over from the multi-agency product that HPSEDC doesn't need can be hidden in a later pass.

Admin console — pipeline, demand analytics, callbacks & support queues.

Test checklist

Candidate

  • Register → see the three registration modes
  • Complete Standard (blue-collar)
  • Complete Professional (degree branch)
  • Try Assisted (callback request → Admin)
  • Upload ID + passport → "ready" + reg. number
  • Toggle Minimal ⇄ Advanced dashboard
  • Open a job → salary band + docs → Apply
  • Check My Application status
  • Open Support (raise issue / check-in)
  • Switch English ⇄ Hindi

HPSEDC Agency

  • All jobs are HPSEDC's; post a new job
  • Move an application Submitted → … → Placed
  • Record a visa/country rejection → jobs hidden
  • Update visa status + travel date

Admin

  • Callbacks queue (Assisted requests)
  • Placement Support queue
  • Grievances list
