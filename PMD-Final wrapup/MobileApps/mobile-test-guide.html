B iOS Setup (iPhone)

1
Install "Expo Go" — Open the App Store on your iPhone → search "Expo Go" → Install.
2
Open Safari (not Expo Go!) and manually type this URL in the address bar:
exps://hirestream-mobile.agentryx.dev
⚠️ Use exps:// not https:// · QR codes do NOT work on iOS — type the URL manually
3
Tap "Open" when iOS asks "Open this page in Expo Go?" — the app will launch automatically.
4
Wait 15–30s for the first load → Login with same credentials above.
📌 Why Safari? iOS Expo Go does not have "Enter URL manually" like Android. Type the URL in Safari — iOS will automatically redirect to Expo Go.

D Troubleshooting

App won't load? Force-close Expo Go and reopen it. Ensure you're on WiFi, not cellular data.
Old build / stale screen? Force-refresh the bundle: Open Expo Go → long-press HireStream in the recent projects list → tap "Remove from list" → "Enter URL manually" → paste exps://hirestream-mobile.agentryx.dev → Connect. Fresh download (~30s). Confirm the build tag in the Applications header now reads v0.4.15.0 or later.
Login fails? Usernames are case-sensitive lowercase. Use mobiletest not MobileTest.
Very slow loading? First load takes 15–30 seconds. Subsequent loads are much faster (~3s).
iOS: QR code doesn't work? Expected. iOS does not support QR for Expo. Type the URL in Safari instead.
Red border on screen? This is an Expo Go dev-mode indicator. Ignore it — it won't appear in production.
"Network Error"? Check internet connection. The app connects to hirestream-stg.agentryx.dev backend.
